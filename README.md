Trabalho da Disciplina EDPA do Mestrado

Instrucoes de uso

Caso ocorra problema de memoria o comando abaixo deve ser executado:
# ulimit -s unlimited

Para compilar o codigo utilize o comando:
# gcc heap.c -o heapexec
# ./heapexec
